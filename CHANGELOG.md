# Changelog

## [0.1.0] — 2026-06-09

### Added
- Layer 1 — Repository Observer: git monitoring, chokidar file watching, hunk-level diff parsing, branch merge detection
- Layer 2 — Change Intelligence Engine: 13-category classification, task completion detection, architecture impact analysis
- Layer 3 — Memory Compression Engine: hierarchical summarization (file→module→component→system), 3-layer memory (short-term/working/long-term), stale fact detection, semantic clustering, duplicate elimination
- Layer 4 — Context Graph: JSON and Neo4j backends, BFS/DFS graph query engine, impact analysis with risk scoring, circular dependency detection, centrality ranking
- Layer 5 — Development Tracking Engine: auto-generated DEVELOPMENT-TRACKING.md, CURRENT-SPRINT.md, NEXT-STEPS.md, ARCHITECTURE-DECISIONS.md, CONTEXT-GRAPH.md
- Layer 6 — Agent Context Builder: 6 agent profiles (Claude Code, Cursor, Archon, Aider, OpenCode, OpenHands), handoff engine, context diff engine, token budget enforcement
- Layer 7 — Knowledge Brain: 128-dim embedding engine with cosine similarity, knowledge validator (5 check dimensions), versioned snapshots with diff, curated ingestion pipeline, semantic search
- Layer 8 — Learning Engine: feedback collector with trend analysis, 5-dimension quality evaluator, compression benchmarker with strategy comparison, A/B testing engine
- Production: multi-repo orchestrator, team memory, governance (RBAC + audit), plugin system, config management
- CLI: 9 commands (run, watch, graph, knowledge, research, feedback, multi-repo, team, config)
- 36 production-grade TypeScript files, strict mode, zero type errors
